Assista o Video do Projeto Aplicativo da Book Store

Simulando uma simpes livraria onde é possível registrar um novo livro, ou uma nova categoria relacionada aos livrosm, o aplicativo faz toda a validação para inserir ou deletar uma categoria baseado na integradidade do seu registro principal. Desenvolvi este projeto com o intuito de criar um simples app com CRUD (Create, Read, Update, Delete), apenas para aplicar meus conhecimentos adquiridos em um bootcamp de aplicações Java, utilizando para criação do front end a biblioteca do Angular 12.
